Getting Started

Welcome to Lofi React Next.js + Strapi template! We would like to thank you, for buy our template - Lofi.

Lofi is a React Next.js, Strapi and Bootstrap 5 based Multipurpose template. This has been built with React and as framework Next.js!

Setup Development Environment

To kickstart the development of the web app with Stax, at first need to setup the react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site https://nodejs.org/en/

  • PostgreSQL

    • Download and install PostgreSQL with PGAdmin.

  • Cloudinary

    • Go to Cloudinary and create an account.

Strapi Version

Strapi Backend please follow the below steps

- Open the lofi-with-strapi-cms/lofi-api and update the lofi-api/config/plugin.js file as the cloudinary credentials.

- Open the pgAdmin server & create the database name like lofi-api and restore the api-demo-data from the downloaded bundle.

- Open the lofi-strapi/config/database.js and update your PostgreSQL credentials & database name.

- Run the command npm install

- Run the command npm run develop

- Go To http://localhost:1337/admin

- Email hello@envytheme.com and Password: EnvyTheme1

- Done

Strapi Front-End please follow the below steps

- Now open the lofi-with-strapi-cms/lofi folder which is in that directory like lofi-with-strapi-cms/lofi and follow like below by opening another command prompt in this folder.

- Run the command npm install

- Run the command npm run dev

- Go To http://localhost:3000/

- Done
